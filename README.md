# Intro to Number Theory · Interactive

**Primes, factors, GCD, LCM — the building blocks of integers**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 1 · Foundation

## What You'll Learn

- Prime numbers and the Sieve of Eratosthenes
- Divisibility rules and factorization
- GCD and LCM (Euclidean algorithm)
- Modular arithmetic basics
- Fun applications: cryptography, magic tricks
